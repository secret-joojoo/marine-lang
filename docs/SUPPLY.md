# 악! 기합 해병이 되고 싶어! 보급병 지침

`악! 기합 해병이 되고 싶어!`의 보급병 지침에 대해 안내합니다.
- 부대에는 `종합창고`, `피복창고`, `물류창고`, `부식창고`, `치장창고`의 총 5개의 창고가 있습니다.
- 각 창고에는 1번 박스부터 256번 박스까지 있습니다.

## 보급병 불러오기
- `보급병! 과업 준비!`로 보급병을 불러옵니다.

## 창고 선언
- `보급병! 창고로 이동`로 배열을 선언합니다. 창고를 선언할 때, 모든 박스는 비어있습니다.
- 창고를 선언하면 `종합창고`, `피복창고`, `물류창고`, `부식창고`, `치장창고` 순서로 박스를 마련합니다.
- 창고를 처음 선언하면 모든 박스는 비어있습니다 (NULL).
```
    보급병! 과업 준비!

    보급병! 창고로 이동
    보급병! 창고로 이동
    보급병! 창고로 이동
```
을 실행하면 `종합창고`, `피복창고`, `물류창고`가 준비됩니다.

## 창고 값 할당 & 변경
- `보급병! (창고) 관리하겠습니다 (정수/변수1) (정수/변수2)`로 `(창고)`의 `(정수/변수1)`번 박스에 `(정수/변수2)`를 할당/변경합니다.
```
    보급병! 과업 준비!

    보급병! 창고로 이동
    보급병! 종합창고 관리하겠습니다 악! 악!악

    보급병! 창고로 이동
    신병 받아라 악!!
    신병 받아라 악!!악
    보급병! 피복창고 관리하겠습니다 아쎄이! 아쎄이!!
```
을 실행하면 `종합창고`의 `1`번 박스에 `10`이 할당되고, `피복창고`의 2번 박스에 `20`이 할당됩니다.

## 창고 값 읽기
- `(변수) (창고) 조사하겠습니다 (정수/변수)`로 `(창고)`의 `(정수/변수)`번 박스 값을 읽어서 `(변수)`에 저장합니다.
```
    보급병! 과업 준비!
    
    보급병! 창고로 이동
    보급병! 종합창고 관리하겠습니다 악! 악!!!!!!악!!!!!
    신병 받아라 악
    아쎄이! 종합창고 조사하겠습니다 악!
    아쎄이 아쎄이! 돌격
    라이라이 차차차
```
을 실행하면, `A`가 출력됩니다.

## 창고 정리
- `보급병! (창고) 정리하겠습니다`로 `(창고)`를 정리할 수 있습니다.
```
    보급병! 과업 준비!

    보급병! 창고로 이동
    보급병! 창고로 이동
    보급병! 창고로 이동
    보급병! 창고로 이동
    보급병! 창고로 이동
    보급병! 종합창고 정리하겠습니다
    보급병! 창고로 이동
```
을 실행하면, 마지막 줄에서 `종합창고`에 박스를 마련합니다.
