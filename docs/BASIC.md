# 악! 기합 해병이 되고 싶어! 공통 지침

`악! 기합 해병이 되고 싶어!`의 공통 지침에 대해 안내합니다.

## 정수 표기
- 0: `악`
- 1 ~ 9: 나타내고 싶은 숫자의 크기만큼 느낌표를 '악' 뒤에 붙입니다.
    - 1: `악!`
    - 2: `악!!`
    - 8: `악!!!!!!!!`
- 두 자리 이상의 정수: 10진수 표기법과 같이 숫자를 연달아 적습니다.
    - 1280: `악!악!!악!!!!!!!!악`
- 음수: 양수 앞에 '아'를 붙입니다.
    - -1: `아악!`
    - -2: `아악!!`
    - -8: `아악!!!!!!!!`
    - -1280: `아악!악!!악!!!!!!!!악`

## 변수 선언
- `신병 받아라 (정수)`의 형식으로 변수를 선언합니다.
- N번째로 선언하면 `아쎄이(N개의 !)`에 정수가 할당됩니다.
```   
    신병 받아라 악!
    신병 받아라 악!!
    신병 받아라 악!!!!!!!!
    신병 받아라 악
```
로 선언한다면, `아쎄이! = 1`, `아쎄이!! = 2`, `아쎄이!!! = 8`, `아쎄이!!!! = 0`가 할당됩니다.

## 덧셈 & 뺄셈
- 덧셈: `(변수1) (변수2) 돌격`을 실행하면, `(변수1)`에는 `(변수1) + (변수2)`가 저장됩니다.
```
    신병 받아라 악!!!
    신병 받아라 악!
    아쎄이! 아쎄이!! 돌격
```
을 실행하면, `아쎄이!`는 `4`를 가집니다.
- 뺄셈: `(변수1) (변수2) 역돌격`을 실행하면, `(변수1)`에는 `(변수1) - (변수2)`가 저장됩니다.
```
    신병 받아라 악!!!
    신병 받아라 악!
    아쎄이! 아쎄이!! 역돌격
```
을 실행하면, `아쎄이!`는 `2`를 가집니다.

## 입출력
- 입력: `헤이빠빠리빠`는 입력된 정수를 `아쎄이`에 할당합니다.
- 출력: `라이라이 차차차`는 `아쎄이`에 있는 정수를 유니코드로 변환하여 출력합니다.
```
    헤이빠빠리빠
    신병 받아라 악!!!악!!
    아쎄이 아쎄이! 돌격
    라이라이 차차차
```
을 실행하고 `65`를 입력하면, `아쎄이`는 `97`이 되어 `a`를 출력합니다.

## 조건문
- `(변수) 여쭤봐도 되겠습니까 필승 (명령문) 받아쓰`는 `(변수)`가 `0`이 아닐 때, `(명령문)`을 실행합니다.
```
    신병 받아라 악!!!!!!악!!!!!
    아쎄이! 여쭤봐도 되겠습니까 필승
        아쎄이 아쎄이! 돌격
        라이라이 차차차
    받아쓰
```
을 실행하면 `A`가 출력됩니다.

## 반복문
- `(변수) 다시 알아보겠습니다 필승 (명령문) 받아쓰`는 `(변수)`가 `0`이 아닐 때 `(명령문)`을 실행하고, 다시 조건을 검사하여 `(명령문)`을 실행합니다.
```
    신병 받아라 악!!!!!
    신병 받아라 악!!!!악!!!!!!!!
    신병 받아라 악!
    신병 받아라 악!악
    아쎄이! 다시 알아보겠습니다 필승
        아쎄이! 아쎄이!! 돌격
        아쎄이 아쎄이! 돌격
        라이라이 차차차
        아쎄이 아쎄이 역돌격
        아쎄이 아쎄이!!!! 돌격
        라이라이 차차차
        아쎄이 아쎄이 역돌격
        아쎄이! 아쎄이!! 역돌격
        아쎄이! 아쎄이!!! 역돌격
    받아쓰
```
을 실행하면 `5\n4\n3\n2\n1`가 출력됩니다.
