# 악! 기합 해병이 되고 싶어! 군악병 지침

`악! 기합 해병이 되고 싶어!`의 군악병 지침에 대해 안내합니다.
- 군악병은 `플루트`, `클라리넷`, `오보에`, `바순`, `색소폰`, `트럼펫`, `트롬본`, `유포늄`, `호른`, `튜바`의 악기를 연주할 수 있습니다.
- 온라인 인터프리터에서 실행하는 것보다, 로컬 환경에서 실행할 때 더 풍부한 소리를 들을 수 있습니다.

## 악보 표기법
- 악보의 양식은 다음과 같습니다.
```
    (BPM)
    (음) ... (음)
    (박자) ... (박자)
    ...
    (음) ... (음)
    (박자) ... (박자)
```
- `(BPM)`에 BPM을 해병 정수로 표기합니다.
- 서로 대응되는 음과 박자는 개수가 항상 일치해야 합니다.
- 박자는 다음과 같이 표기합니다.

    |**1박**|**2박**|**3박**|**4박**|
    |---|---|---|---|
    |악!|악!!|악!!!|악!!!!|
    |**1/2박**|**1/4박**|**1/8박**|**1/16박**|
    |아악!|아악!!|아악!!!|아악!!!!|

- 음은 `도 도# 레 레# 미 파 파# 솔 솔# 라 라# 시`으로 표기하며, N옥타브의 음은 뒤에 !를 N개 더 붙입니다. 쉼표는 `악`으로 표기합니다.
    - 0옥타브 솔: `솔`
    - 2옥타브 도#: `도#!!`
    - 3옥타브 시b: `라#!!!`
    - 쉼표: `악`

## 군악대 불러오기
- `군악대 도열 (정수)`로 `(정수)`명의 군악병을 한 자리에 위치시킵니다.
- 군악대는 `군악대!`라고 부릅니다.
- N번째 군악병은 `군악병(N개의 !)`로 부릅니다.
- 최대 한 군악대만 동시에 불러올 수 있습니다.

## 군악대 악기 설정
- `군악대! (악기) ... (악기) 준비`로 군악대의 악기를 설정합니다.
- `군악대!`의 인원수에 맞추어 악기를 설정해야 합니다.
```
    군악대 도열 악!!!!!
    군악대! 플루트 클라리넷 오보에 바순 색소폰 준비
```
을 실행하면, `군악병!`은 `플루트`, `군악병!!`은 `클라리넷`, `군악병!!!`은 `오보에`, `군악병!!!!`은 `바순`, `군악병!!!!!`은 `색소폰` 연주를 준비합니다.

## 군악대 악보 저장
- `소중히 간직할 수 있도록 필승 (악보) 받아쓰`로 `(악보)`를 저장할 수 있습니다.
- N번째 저장한 악보는 `숙지완료하였습니다(N개의 !)`에 저장됩니다.

## 군악병 악보 설정
- `(군악병) 제대로 숙지할 수 있도록 필승 (저장된 악보) 받아쓰`로 `(군악병)`이 `(저장된 악보)`로 숙지합니다.
```
    군악대 도열 악!!
    군악대! 색소폰 바순 준비
    소중히 간직할 수 있도록 필승
        악!!!!!!!!!악
        솔 솔 라 라 솔 솔 미 악
        악! 악! 악! 악! 악! 악! 악! 악!
        솔 솔 미 미 레 악
        악! 악! 악! 악! 악! 악!!!
        솔 솔 라 라 솔 솔 미 악
        악! 악! 악! 악! 악! 악! 악! 악!
        솔 미 레 미 도
        악! 악! 악! 악! 악!!!!
    받아쓰
    군악병! 제대로 숙지할 수 있도록 필승
        숙지완료하였습니다!
    받아쓰
```
를 실행하면 `군악병!`은 90 BPM의 '학교 종이 땡땡땡' 악보를 숙지합니다.

## 군악대 대기
- `군악대! 대기하겠습니다 (정수)`으로 연주 시작 직전 `(정수)`초 만큼 코드를 실행하지 않고 대기합니다.

## 군악대 연주
- `군악대! 연주 시작`으로 `군악대!` 내의 모든 군악병들이 동시에 연주를 시작합니다.
- 연주를 시작하기 전에 모든 군악병들이 악보를 숙지해야 합니다.
- 군악병들은 가장 긴 악보를 가진 군악병의 연주가 끝날 때까지 대기합니다.
- 연주가 모두 끝날 때까지 다음 줄의 코드가 실행되지 않습니다.

## 군악대 퇴장
- `군악대! 총원 헤쳐`로 `군악대!`가 퇴장합니다.
- `군악대!`, 모든 `군악병`에 대한 정보가 삭제됩니다.
- 저장된 악보에 대한 정보는 유지됩니다.
