# 악! 기합 해병이 되고 싶어! 에러 케이스

`악! 기합 해병이 되고 싶어!`의 에러 케이스를 다룹니다.

## 목차

## 기본 인터프리터

### 정수 표기
- `악`, `!`, `아` 외의 문자가 있는 경우
- `악` 뒤에 `!`가 10개 이상 붙는 경우
- `악` 뒤에 `아`가 오는 경우

### 변수 선언
- `신병 받아라` 뒤에 정수가 오지 않는 경우

### 덧셈 & 뺄셈
- 선언되지 않은 변수를 사용하는 경우

### 입출력
- `헤이빠빠리빠`로 입력 받을 때, 정수 외의 입력이 들어오는 경우
- `라이라이 차차차`로 출력할 때, `아쎄이`에 유니코드로 변환 불가능한 수가 들어있는 경우

### 조건문
- `여쭤봐도 되겠습니까` 앞에 선언되지 않은 변수가 있는 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우

### 반복문
- `다시 알아보겠습니다` 앞에 선언되지 않은 변수가 있는 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우
