# 악! 기합 해병이 되고 싶어! 에러 케이스

`악! 기합 해병이 되고 싶어!`의 에러 케이스를 다룹니다.

## 기본 문법

### 정수 표기
- `악`, `!`, `아` 외의 문자가 있는 경우
- `악` 뒤에 `!`가 10개 이상 붙는 경우
- `악` 뒤에 `아`가 오는 경우

### 변수 선언: `신병 받아라 (정수)`
- `신병 받아라` 뒤에 정수가 오지 않는 경우

### 덧셈 & 뺄셈: `(변수1) (변수2) (역)돌격`
- 선언되지 않은 변수를 사용하는 경우

### 입출력: `헤이빠빠리빠` & `라이라이 차차차`
- `헤이빠빠리빠`로 입력 받을 때, 정수 외의 입력이 들어오는 경우
- `라이라이 차차차`로 출력할 때, `아쎄이`에 유니코드로 변환 불가능한 수가 들어있는 경우

### 조건문: `(변수) 여쭤봐도 되겠습니까 필승 (명령문) 받아쓰`
- `여쭤봐도 되겠습니까` 앞에 선언되지 않은 변수가 있는 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우

### 반복문: `(변수) 다시 알아보겠습니다 필승 (명령문) 받아쓰`
- `다시 알아보겠습니다` 앞에 선언되지 않은 변수가 있는 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우

## 보급병 지침

### 보급병 불러오기: `보급병! 과업 준비!`
- 보급병을 부르지 않고 보급병 함수를 사용하는 경우

### 창고 선언: `보급병! 창고로 이동`
- 비어있는 창고가 없을 때 창고를 선언하는 경우

### 창고 값 할당 & 변경: `보급병! (창고) 관리하겠습니다 (정수/변수1) (정수/변수2)`
- 선언되지 않은 창고를 관리하는 경우
- 선언되지 않은 변수를 사용하는 경우
- 1번 ~ 256번 박스 외의 박스를 호출하는 경우

### 창고 값 읽기: `(변수) (창고) 조사하겠습니다 (정수/변수)`
- 선언되지 않은 창고를 조사하는 경우
- 선언되지 않은 변수를 사용하는 경우
- 1번 ~ 256번 박스 외의 박스를 호출하는 경우
- 빈 박스를 호출하는 경우

### 창고 정리: `보급병! (창고) 정리하겠습니다`
- 사용 중인 창고가 없을 때 정리하는 경우

## 군악병 지침

### 군악대 불러오기: `군악대 도열 (정수)`
- `(정수)`가 1보다 작은 경우
- `군악대 도열` 뒤에 정수가 오지 않는 경우

### 군악대 악기 설정: `군악대! (악기) ... (악기) 준비`
- 악기 개수와 군악대 인원수가 일치하지 않는 경우
- 군악대 악기가 아닌 악기를 설정하는 경우
- 군악대를 불러오지 않고 악기를 설정하는 경우

### 군악대 악보 저장: `소중히 간직할 수 있도록 필승 (악보) 받아쓰`
- 악보 양식에 어긋나는 경우
- BPM이 1보다 작은 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우

### 군악대 악보 설정: `(군악병) 제대로 숙지할 수 있도록 필승 (저장된 악보) 받아쓰`
- 저장되지 않은 악보를 숙지하는 경우
- 없는 군악병을 부르는 경우
- `필승`과 `받아쓰`의 짝이 맞지 않는 경우
- `필승`과 `받아쓰`가 없는 경우
- `필승`과 `받아쓰` 사이의 명령문의 들여쓰기가 되지 않은 경우

### 군악대 대기: `군악대! 대기하겠습니다 (정수)`
- `(정수)`가 1보다 작은 경우
- 다음 명령어가 `군악대! 연주 시작`이 아닌 경우

### 군악대 연주: `군악대! 연주 시작`
- 악보를 숙지하지 않은 군악병이 있을 경우

### 군악대 퇴장: `군악대! 총원 헤쳐`
- 군악대를 불러오지 않은 경우
